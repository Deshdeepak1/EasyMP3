# EasyMP3
Downloading latest songs from terminal made easy!

Python dependencies - bs4, requests, tqdm

# Usage
 1. python3 -m venv venv
 2. . ./venv/bin/activate
 3. pip3 install -r requirements.txt
 4. python3 song.py
